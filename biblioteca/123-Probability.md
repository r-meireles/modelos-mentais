# [[Probability]]

> **Teste de Pertinência:** *Você está tentando prever resultado de evento aleatório ou incerto e precisa quantificar quão provável é que algo aconteça? Se a resposta for "Sim", este modelo é a ferramenta correta para o momento.*

## 1. Definição e Mecanismo
**O que é:**
Probability (Probabilidade) é probabilidade de que algo aconteça. Na maioria dos casos, não podemos dizer, com certeza, qual resultado ocorrerá, mas frequentemente podemos prever quão provável é dado resultado. Probabilidade é medida numérica de chance ou likelihood de evento ocorrer. Varia de 0 (impossível) a 1 (certo), ou de 0% a 100%.

**Explicação:**
Este modelo funciona porque reconhece que muitos eventos no mundo real são incertos - não podemos prever com certeza o que acontecerá, mas podemos quantificar quão provável é que diferentes resultados ocorram. Probabilidade fornece estrutura matemática para lidar com incerteza. Isto funciona porque reconhece que através de usar probabilidade, você pode fazer previsões informadas, avaliar riscos, e tomar decisões sob incerteza.

A lógica central é que através de usar probabilidade, você pode quantificar incerteza e fazer previsões informadas sobre resultados futuros. Isto funciona porque reconhece que diferentes métodos de calcular probabilidade (clássico, empírico, subjetivo) são apropriados para diferentes situações.

Funciona porque:
- **Quantifica incerteza:** Fornece forma de quantificar incerteza sobre resultados futuros.
- **Permite previsões:** Permite fazer previsões informadas sobre quão provável é que diferentes resultados ocorram.
- **Avalia riscos:** Ajuda a avaliar riscos - probabilidade de diferentes resultados ajuda a entender risco.
- **Informa decisões:** Informa tomada de decisão sob incerteza - você pode escolher ações que maximizam probabilidade de resultados desejados.

**Métodos principais:**
- **Probabilidade Clássica (Teórica):** Quando cada resultado é igualmente provável - P(E) = (número de resultados favoráveis) ÷ (número total de resultados).
- **Probabilidade Empírica (Baseada em Frequência):** Baseada em observar muitas tentativas repetidas e estimar probabilidades de frequências relativas.
- **Probabilidade Subjetiva:** Baseada em julgamento pessoal ou crença sobre quão provável é evento.

## 2. Pressupostos Fundamentais
*O modelo só é válido se estas premissas forem verdadeiras:*
* **Pressuposto 1:** Há evento aleatório ou incerto cuja probabilidade você quer calcular.
* **Pressuposto 2:** Você pode identificar todos os resultados possíveis (espaço amostral).
* **Pressuposto 3:** Você pode atribuir probabilidades a diferentes resultados (soma deve ser 1).
* **Pressuposto 4:** Eventos são bem definidos e você pode determinar se ocorreram ou não.

## 3. Contexto de Aplicação

### Quando Usar (Luz Verde)
* Situações onde o modelo produz boas decisões rapidamente:*
- **Prever resultados:** Ao prever resultados de eventos aleatórios ou incertos, usar probabilidade para quantificar quão provável é que diferentes resultados ocorram.
- **Avaliar riscos:** Ao avaliar riscos, usar probabilidade para entender quão provável é que diferentes resultados (bons ou ruins) ocorram.
- **Tomar decisões sob incerteza:** Ao tomar decisões sob incerteza, usar probabilidade para informar escolhas.
- **Entender aleatoriedade:** Ao entender aleatoriedade, usar probabilidade para quantificar incerteza.

**Domínios Típicos:** Estatística, Probabilidade, Tomada de Decisão, Análise de Risco, Jogos de Azar.

### Quando Evitar (Luz Vermelha)
* Situações onde o modelo induz ao erro ou simplifica demais a realidade:*
- **Eventos determinísticos:** Se evento é determinístico (resultado é certo), probabilidade não é apropriada.
- **Probabilidades subjetivas incorretas:** Se probabilidades subjetivas estão incorretas (baseadas em julgamento pobre), análises estarão incorretas.
- **Não considerar dependências:** Não considerar dependências entre eventos - eventos podem não ser independentes.
- **Ignorar contexto:** Ignorar contexto - probabilidades podem mudar dependendo de contexto ou informação disponível.

## 4. Zona de Risco (Diagnóstico)

### Uso Incorreto Comum
* Como amadores costumam distorcer esse modelo:*
- **Confundir probabilidade com certeza:** Confundir probabilidade alta com certeza - mesmo eventos muito prováveis podem não acontecer.
- **Ignorar dependências:** Ignorar que eventos podem não ser independentes - probabilidade de um pode afetar probabilidade de outro.
- **Usar probabilidades incorretas:** Usar probabilidades incorretas - baseadas em julgamento pobre ou dados incorretos.
- **Não atualizar probabilidades:** Não atualizar probabilidades à luz de nova evidência - usar Teorema de Bayes.

### Sinais de que Você Saiu do Modelo
* **Indicador Comportamental:** 
  - Você está confundindo probabilidade alta com certeza.
  - Você não está considerando dependências entre eventos.
  - Você está usando probabilidades incorretas.
  - Você não está atualizando probabilidades à luz de nova evidência.

* **Evidência Prática:** 
  - Suas previsões estão incorretas porque você usou probabilidades incorretas.
  - Você não está capturando realidade porque ignorou dependências ou não atualizou probabilidades.

## 5. Passos de Pensamento (Algoritmo)
1. **Identifique evento:** Identifique evento aleatório ou incerto cuja probabilidade você quer calcular.

2. **Identifique espaço amostral:** Identifique espaço amostral - conjunto de todos os resultados possíveis.

3. **Escolha método apropriado:** Escolha método apropriado para calcular probabilidade:
   - **Clássico:** Se cada resultado é igualmente provável
   - **Empírico:** Se você tem dados de tentativas repetidas
   - **Subjetivo:** Se você está usando julgamento pessoal

4. **Calcule probabilidade:** Calcule probabilidade usando método apropriado.

5. **Considere dependências:** Considere se eventos são independentes ou dependentes - probabilidade de um pode afetar probabilidade de outro.

6. **Use para previsão:** Use probabilidade para fazer previsões informadas sobre quão provável é que diferentes resultados ocorram.

7. **Use para avaliação de risco:** Use probabilidade para avaliar riscos - probabilidade de diferentes resultados ajuda a entender risco.

8. **Atualize com nova evidência:** Atualize probabilidades à luz de nova evidência - use Teorema de Bayes se apropriado.

9. **Monitore e ajuste:** Após fazer previsões baseadas em probabilidade, monitore se estão funcionando como esperado. Ajuste se necessário.

10. **Use apropriadamente:** Use probabilidade para quantificar incerteza e fazer previsões informadas, mas reconheça limitações - probabilidade não é certeza e pode mudar com nova informação.
