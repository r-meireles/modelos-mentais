# [[Heredity]]

> **Teste de Pertinência:** *Você está tentando entender como características ou traços são passados de uma geração ou iteração para próxima, ou está tentando aplicar princípios de herança para entender transmissão de características? Se a resposta for "Sim", este modelo é a ferramenta correta para o momento.*

## 1. Definição e Mecanismo
**O que é:**
Heredity (Hereditariedade), também conhecida como herança, é processo biológico através do qual pais passam traços (características) para seus descendentes. Genes são unidades de hereditariedade feitas de DNA, localizadas em cromossomos. Cada gene carrega informação para traços específicos. Alelos são diferentes versões de um gene - para cada gene, indivíduo herda dois alelos, um de cada pai.

**Explicação:**
Este modelo funciona porque reconhece que traços são herdados através de genes. Genótipo é composição genética (quais alelos alguém carrega), enquanto fenótipo são traços mensuráveis (como cor de olhos, altura), que resultam de genótipo mais influências ambientais. Herança pode seguir padrões simples (Mendelianos) onde um gene controla traço, ou padrões complexos onde múltiplos genes e fatores ambientais contribuem.

A lógica central é que através de herança, características são passadas de uma geração para próxima. Isto permite que traços que aumentam fitness sejam preservados e passados adiante. No entanto, herança não é determinística - ambiente também influencia como traços se desenvolvem. Genes fornecem blueprint, mas fatores ambientais (nutrição, exposição, estilo de vida) influenciam se e como traços se desenvolvem.

Funciona porque:
- **Transmite características:** Através de herança, características são passadas de uma geração para próxima.
- **Preserva traços benéficos:** Traços que aumentam fitness são preservados e passados adiante.
- **Cria variação:** Diferentes combinações de alelos criam variação em traços.
- **Interage com ambiente:** Genes fornecem blueprint, mas ambiente influencia como traços se desenvolvem.

**Padrões principais de herança:**
- **Autossômico dominante:** Apenas um alelo mutante é suficiente para traço ser expresso
- **Autossômico recessivo:** Duas cópias de alelo mutante são necessárias para expressão
- **Ligado ao X:** Genes localizados no cromossomo X
- **Poligênico:** Muitos genes contribuem para traço
- **Multifatorial:** Muitos genes mais fatores ambientais contribuem

## 2. Pressupostos Fundamentais
*O modelo só é válido se estas premissas forem verdadeiras:*
* **Pressuposto 1:** Traços são herdados através de genes - unidades de hereditariedade feitas de DNA.
* **Pressuposto 2:** Indivíduos herdam dois alelos para cada gene, um de cada pai.
* **Pressuposto 3:** Genótipo (composição genética) mais influências ambientais determinam fenótipo (traços observáveis).
* **Pressuposto 4:** Herança pode seguir padrões simples (um gene) ou complexos (múltiplos genes, fatores ambientais).

## 3. Contexto de Aplicação

### Quando Usar (Luz Verde)
* Situações onde o modelo produz boas decisões rapidamente:*
- **Entender transmissão de características:** Ao entender como características são transmitidas, aplicar princípios de hereditariedade.
- **Prever traços:** Ao prever traços em descendentes, usar padrões de herança conhecidos.
- **Entender variação:** Ao entender variação, reconhecer que diferentes combinações de alelos criam variação.
- **Aplicar a sistemas não biológicos:** Ao aplicar a sistemas não biológicos (ex: cultura, conhecimento), reconhecer que características podem ser "herdadas" através de transmissão.
- **Entender interação gene-ambiente:** Ao entender como traços se desenvolvem, reconhecer que genes e ambiente interagem.

**Domínios Típicos:** Biologia, Genética, Medicina, Antropologia, Cultura, Conhecimento.

### Quando Evitar (Luz Vermelha)
* Situações onde o modelo induz ao erro ou simplifica demais a realidade:*
- **Quando não há herança:** Se não há mecanismo de herança (características não são passadas), modelo não se aplica.
- **Determinismo genético:** Não usar modelo para justificar determinismo genético - ambiente também importa.
- **Não como justificativa:** Não usar modelo como justificativa para discriminação ou desigualdade.
- **Não assumir simplicidade:** Não assumir que todos os traços seguem padrões simples de herança - muitos são complexos.

## 4. Zona de Risco (Diagnóstico)

### Uso Incorreto Comum
* Como amadores costumam distorcer esse modelo:*
- **Determinismo genético:** Assumir que genes determinam tudo, ignorando papel de ambiente.
- **Justificar desigualdade:** Usar modelo como justificativa para desigualdade ou discriminação.
- **Assumir simplicidade:** Assumir que todos os traços seguem padrões simples de herança.
- **Ignorar interação gene-ambiente:** Ignorar que genes e ambiente interagem para determinar traços.

### Sinais de que Você Saiu do Modelo
* **Indicador Comportamental:** 
  - Você está assumindo que genes determinam tudo.
  - Você está usando modelo como justificativa para desigualdade.
  - Você está assumindo que todos os traços seguem padrões simples.
  - Você não está reconhecendo interação gene-ambiente.

* **Evidência Prática:** 
  - Suas previsões de traços estão incorretas porque você ignorou ambiente.
  - Você está usando modelo para justificar comportamento inapropriado.

## 5. Passos de Pensamento (Algoritmo)
1. **Identifique características relevantes:** Identifique características ou traços que você está tentando entender ou prever.

2. **Avalie se há herança:** Determine se há mecanismo de herança - características são passadas de uma geração ou iteração para próxima?

3. **Identifique padrão de herança:** Se aplicável, identifique padrão de herança - simples (um gene) ou complexo (múltiplos genes, fatores ambientais)?

4. **Considere genótipo e fenótipo:** Reconheça diferença entre genótipo (composição genética ou "blueprint") e fenótipo (traços observáveis).

5. **Considere interação gene-ambiente:** Reconheça que genes fornecem blueprint, mas ambiente influencia como traços se desenvolvem.

6. **Use para prever:** Se aplicável, use padrões de herança conhecidos para prever traços em descendentes ou próximas iterações.

7. **Aplique a sistemas não biológicos:** Se aplicável, aplique princípios de herança a sistemas não biológicos (ex: cultura, conhecimento) - como características são transmitidas?

8. **Reconheça limitações:** Reconheça que herança não é determinística - ambiente também importa.

9. **Monitore e ajuste:** Após aplicar modelo, monitore se está funcionando como esperado. Ajuste se necessário.

10. **Use apropriadamente:** Use modelo para entender transmissão de características, não como justificativa para desigualdade ou discriminação.
